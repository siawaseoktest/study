平方完成の解説

平方完成は、2次方程式や2次関数の式を変形して「平方の形」にする方法です。この手法を使うと、グラフの頂点の座標を簡単に求めたり、解の公式を導出したりできます。

一般的な2次式

2次式の一般的な形は次のようになります。

ax2 + bx + c （ここで a ≠ 0）

平方完成の手順

  1. 係数 a を外に出す
    まず、a を括り出します。
    ax2 + bx + c = a(x2 + (b/a)x) + c
  2. x2 + (b/a)x を平方の形にする
    (x + p)2 = x2 + 2px + p2 の形を利用して、平方を作ります。
    x2 + (b/a)x = (x + b/2a)2 - (b/2a)2
  3. 元の式に代入して整理する
    ax2 + bx + c = a[(x + b/2a)2 - (b/2a)2] + c
    = a(x + b/2a)2 - a(b/2a)2 + c
  4. 定数項をまとめる
    ax2 + bx + c = a(x + b/2a)2 + (c - b2/4a)

例: x2 + 4x + 5 の平方完成

  1. 係数を確認
    a = 1, b = 4, c = 5。係数 a はそのまま。
  2. 平方の形に変形
    x2 + 4x = (x + 2)2 - 22 = (x + 2)2 - 4
  3. 定数項を足す
    x2 + 4x + 5 = (x + 2)2 - 4 + 5 = (x + 2)2 + 1

平方完成の結果

x2 + 4x + 5 = (x + 2)2 + 1

これにより、頂点の座標は (-2, 1)、グラフの形が明確になります。

応用

  • 2次関数の頂点を求める
  • 2次方程式の解を求める
  • 物理学や経済学の最適化問題
